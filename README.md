## 🚀 How to Deploy on GitHub Pages (Free Hosting)

### Step 1 — Create a GitHub Repository
1. Go to [github.com](https://github.com) → **New Repository**
2. Name it: `portfolio` (or `shubhamgupta.github.io` for root domain)
3. Set to **Public**
4. Click **Create Repository**

### Step 2 — Upload Files
**Option A — Upload via browser:**
1. Open your new repo
2. Click **Add file → Upload files**
3. Drag and drop `index.html` and `README.md`
4. Commit changes

**Option B — Git CLI:**
```bash
git init
git add .
git commit -m "Initial portfolio deploy"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/portfolio.git
git push -u origin main
```

### Step 3 — Enable GitHub Pages
1. Go to repo **Settings → Pages**
2. Under **Source**, select `main` branch → `/ (root)`
3. Click **Save**
4. Your site will be live at: `https://YOUR_USERNAME.github.io/portfolio`

> ⏱️ Takes 1–2 minutes to go live after enabling.
